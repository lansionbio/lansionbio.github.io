二. 主界面

本系统主要分为5个部分，分别为：人事、设备、排班、报表、系统。如图2 所示：

1. 人事

人事部分只使用以下功能，如图2-1 所示：
【机构信息】：部门。
【人员信息】：人员。
【人事变更】：人员离职。

1.1 部门

可操作新增、查看、编辑、删除、导出、导入，如图2-1-1 所示：

1.2 人员

可操作导出、导入、新增、删除、对选取数据进行操作、修改验证方式、修改归属地，如图2-1-2 所示：

1.3 人员离职

可以查看已离职人员、新增离职人员和对已离职的人员进行离职恢复等操作，如图2-1-3 所示：

2. 设备

设备只使用以下功能，如图2-2 所示：
【设备管理】：设备管理。
【常用功能】：服务器下发命令。
【日志】：设备操作日志。

2.1 设备管理

可操作导出、导入、新增、删除、对选取数据进行操作、设置归属。如图2-2-1 所示：
日常无需使用该部分，系统自动同步。

2.2 服务器下发命令

可查看、导出服务器生成的命令内容，用于系统排错。如图2-2-2 所示：

2.3 设备操作日志

设备操作日志记录的是人员对设备进行的操作记录，例如开机、关机、报警、进入菜单等。如图2-2-3 所示：

3. 排班

排班部分只使用以下功能，如图3-1 所示：
【班次】：时段管理、班次管理、人员排班。
【休假设置】：节假日。
【基本设置】：考勤规则。
